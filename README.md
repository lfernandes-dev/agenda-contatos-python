# Agenda de Contatos em Python

Projeto simples feito para praticar lógica de programação.

## Funções do sistema

- Adicionar contato
- Editar contato
- Pesquisar contato
- Listar contatos
- Apagar contatos

## Tecnologias usadas

- Python
- Lista
- Dicionário
- Funções
- Loop while

## Como rodar o projeto

Abra o terminal na pasta e rode:

python agenda.py
